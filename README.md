# focusflow-landing-page
Landing page moderna e totalmente responsiva criada para o FocusFlow, um aplicativo fictício que incentiva a concentração recompensando o usuário com o plantio de árvores reais. Projeto desenvolvido para praticar HTML, CSS e Bootstrap 5.

Uma landing page moderna e totalmente responsiva para um aplicativo fictício de produtividade. O conceito: a cada 25 minutos de foco, você planta uma árvore real.

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização customizada (gradientes, animações, hover)
- **Bootstrap 5.3** - Grid responsivo, componentes e utilitários
- **Unsplash** - Imagem gratuita de alta qualidade

## ✨ Funcionalidades Implementadas

- ✅ **Navbar Responsiva** - Menu hamburguer em dispositivos móveis
- ✅ **Hero Section** - Banner principal com call-to-action
- ✅ **Grid de Cards** - 3 colunas no desktop, 2 no tablet, 1 no mobile
- ✅ **Design Totalmente Responsivo** - Adapta-se a qualquer tamanho de tela
- ✅ **Efeitos Hover** - Feedback visual ao passar o mouse nos cards
- ✅ **Gradiente CSS** - Fundo suave na seção principal

## 📱 Comportamento Responsivo

| Dispositivo | Largura | Cards por Linha | Menu |
| :--- | :--- | :--- | :--- |
| Mobile | < 768px | 1 | Hamburguer |
| Tablet | 768px - 992px | 2 | Normal/Hamburguer |
| Desktop | > 992px | 3 | Normal |
